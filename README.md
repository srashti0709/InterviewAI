# 🤖 InterviewAI

**InterviewAI** is an AI-powered mock interview platform designed to help users prepare for technical and HR interviews through realistic AI-driven interview simulations. It provides personalized questions, voice interaction, resume-based interviews, detailed performance analysis, and downloadable reports to improve interview readiness.

---

# ✨ Features

### 🎯 AI Mock Interviews
- Role and experience-based interview questions
- Intelligent follow-up questions generated by AI
- Personalized interview experience

### 🎤 Voice Interview Mode
- AI asks questions using speech synthesis
- Browser speech recognition captures user responses
- Natural conversational interview experience

### 📄 Resume-Based Interviews
- Upload PDF/DOC resumes
- AI analyzes projects, skills, and experience
- Generates customized interview questions

### 👤 AI Interviewer Selection
- Choose between Male or Female AI interviewer
- Animated AI interviewer experience

### 📹 Live Camera Preview
- Real-time webcam during interviews
- Simulates actual interview environment

### ⏱️ Interview Timer
- Circular countdown timer for every question
- Helps practice time management

### 📊 AI Performance Report
Receive detailed feedback including:
- Overall Interview Score
- Communication Skills
- Confidence Score
- Correctness Analysis
- Individual Question Feedback
- Performance Charts

### 📑 PDF Report Download
- Export complete interview reports as PDF
- Share or review performance anytime

### 📚 Interview History
- Access previous interviews
- View complete reports anytime

### 💳 Credit System
- Free and Premium interview credits
- Razorpay payment integration

### 🔐 Secure Authentication
- Google Sign-In using Firebase Authentication

# 🚀 Getting Started

## Prerequisites

Before running the project, make sure you have:

- Node.js (v18 or later)
- npm
- MongoDB Database
- Firebase Project (Google Authentication Enabled)
- Razorpay Account

---

## Clone Repository

```bash
git clone https://github.com/your-username/InterviewAI.git

cd InterviewAI
```

---

## Backend Setup

```bash
cd server

npm install
```

Create a `.env` file inside the **server** folder and configure:

```env
MONGODB_URI=

JWT_SECRET=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=

FIREBASE_PROJECT_ID=

...
```

Start the backend server:

```bash
npm start
```

Backend runs on:

```
http://localhost:8000
```

---

## Frontend Setup

```bash
cd client

npm install
```

Create a `.env` file inside **client**:

```env
VITE_RAZORPAY_KEY_ID=
```

Configure Firebase credentials in:

```
client/src/utils/firebase.js
```

Run the frontend:

```bash
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```

# 📡 REST API Endpoints

| Method | Endpoint | Description |
|---------|----------|-------------|
| GET | `/api/user/current-user` | Get Current User |
| POST | `/api/auth/google` | Google Authentication |
| GET | `/api/auth/logout` | Logout |
| POST | `/api/interview/resume` | Upload Resume |
| POST | `/api/interview/generate-questions` | Generate AI Questions |
| POST | `/api/interview/submit-answer` | Submit Interview Answer |
| POST | `/api/interview/finish` | Generate Final Report |
| GET | `/api/interview/get-interview` | Interview History |
| GET | `/api/interview/report/:id` | Interview Report |
| POST | `/api/payment/order` | Create Razorpay Order |
| POST | `/api/payment/verify` | Verify Payment |
